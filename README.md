## Особенности парсинга

Приложение использует несколько методов для извлечения количества подписчиков:

1. **Поиск в store данных** - извлекает данные из JavaScript store
2. **Регулярные выражения** - ищет `likesCount` в исходном коде
3. **Анализ script тегов** - парсит JavaScript код

### Отладка парсера

При первом запуске приложение сохраняет HTML страницы в файл `debug_page.html` и выводит в лог все найденные упоминания `likesCount`.

Если парсер не находит данные, вы можете:

1. Проверить файл `debug_page.html` для ручного анализа
2. Настроить регулярные выражения в `parser.py` под конкретную структуру страницы
3. Увеличить уровень логирования для детальной отладки
